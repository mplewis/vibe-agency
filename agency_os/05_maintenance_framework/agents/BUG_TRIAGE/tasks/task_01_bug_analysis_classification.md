# Task: Bug Analysis & Classification

## Objective
Parse bug_report.json, classify severity/category, assess impact.

## Input Artifacts
- `bug_report.json`
- `MAINTENANCE_triage_rules.yaml`

## Process
1. Parse and understand bug_report.json
2. Classify severity (P1_Critical, P2_High, etc.)
3. Classify category (Security, Performance, Functional)
4. Assess impact on users/data/system
5. Evaluate reproducibility

## Output
Bug classification result

## Success Criteria
- ✅ Severity assigned correctly
- ✅ Category assigned
- ✅ Impact assessed
- ✅ Reproducibility evaluated
